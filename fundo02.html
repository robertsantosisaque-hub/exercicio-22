Podemos fazer de varias maneiras o ajuste de fundos em nosso site

Existem várias formas de ajustar os fundos em nosso site. Uma delas é usando imagens repetidas com o background-repeat: repeat;

Aumentando e diminuindo o tamanho das imagens com o background-size: contain ou background-size: cover;

Modelos de fundo

Podemos usar modelos de fundo para criar um visual mais interessante e personalizado para o nosso site. Existem muitos sites que oferecem modelos de fundo gratuitos, como o Subtle Patterns e o Transparent Textures. Porém as imagens usando uma url de sites pode haver problemas futuros se o site for desativado ou mudar de localização.
